To use this Guide, you will need Fundamental knowledge of JavaScript.
You will need the most recent version of Webpack4 ready to be installed. Ensure you have the latest version of Webpack 4 installed. As of the date of this publication, the most current release is Webpack version 4.29.6. Visit https://webpack.js.org to find the most current version.
You will need the use of command line using Command Prompt
Ensure you've installed the latest version of NodeJS otherwise, you may encounter a lot of issues in an older version.
Node.js,  the JavaScript run-time environment
You will need NPM installed on your local machine. Webpack and Webpack 4 runs in the Node.js environment, this is why we need its package manager—NPM.

Install the NPM package manager which you will use with Wepback 4.

mkdir webpack4 && cd webpack4
npm init -y
npm install webpack webpack-cli --save-dev

npm install --save-dev webpack
npm install --save-dev webpack@<version>

npm install --save-dev webpack-cli


npm install --global webpack
